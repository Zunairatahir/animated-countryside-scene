# 🌾 Animated Countryside Scene

> A beautifully animated countryside landscape built using **HTML5 Canvas**, **CSS3**, and **Vanilla JavaScript**. The project features a rotating windmill, moving wagon, flying birds, animated grass, drifting clouds, and a dynamic daytime sky.

---

## 📖 Overview

This project demonstrates how to create an animated 2D landscape using the HTML5 Canvas API without relying on any external libraries or frameworks. Every visual element is drawn and animated programmatically using JavaScript.

---

# ✨ Features

- 🌤️ Dynamic daytime sky
- ☀️ Animated sun
- ☁️ Moving clouds
- 🐦 Flying birds with wing animation
- 🌾 Procedurally animated grass
- 🌳 Rolling hills
- 🌬️ Rotating windmill
- 🛒 Moving wooden wagon
- 🎨 Gradient lighting and shadows
- 🔄 Smooth animation using `requestAnimationFrame()`
- 🚀 Built entirely with Vanilla JavaScript

# ⚙️ How It Works

The project is built around the HTML5 Canvas API.

The JavaScript continuously redraws the scene using `requestAnimationFrame()`, creating smooth animations for:

- Windmill rotation
- Wagon movement
- Cloud drifting
- Bird flight
- Grass swaying
- Dynamic lighting
